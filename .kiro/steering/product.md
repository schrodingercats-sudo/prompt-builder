# Product Overview

**Promptify** is an AI-powered prompt optimization tool that helps users create better, more detailed prompts for building web applications. 

## Core Features
- **Prompt Enhancement**: Uses Google's Gemini AI to analyze and improve user prompts
- **Multi-modal Input**: Supports both text and image inputs for context
- **AI Model Integration**: Designed to work with various AI coding assistants (Lovable AI, Cursor, v0, Replit, Bolt)
- **User Management**: Simple email-based authentication with local storage
- **Credit System**: Rate limiting with daily credit resets (2 credits per day for regular users)
- **Community Features**: Shared prompt library and user-generated content

## Target Users
Developers and creators who want to generate more effective prompts for AI-powered development tools.
